## 1. 사용자 여정 (User Journey)


### 1.1 신규 사용자 여정

**진입 경로 1: YouTube (신규 유저층 확보)**

    YouTube 접속  →  쇼츠/하이라이트  →  풀 영상 관심?  →  WSOPTV 이동  →  구독


**진입 경로 2: WSOPTV 직접 접속**

    랜딩페이지 접속  →  콘텐츠 미리보기  →  회원가입  →  구독


💡 **투트랙 전략**
YouTube는 신규 유저층이 많아 무료 콘텐츠(쇼츠, 하이라이트)로 유입을 유도합니다.
WSOPTV는 구독자 전용 풀 에피소드 서비스입니다.


### 1.2 기존 사용자 여정

    앱/웹 접속  →  로그인  →  홈 화면
                              ↓
                         시청 선택
                        ↙        ↘
              이어보기              새 콘텐츠
                 ↓                    ↓
         중단 지점부터 재생      브라우징  →  에피소드 선택  →  시청 시작
                                                                 ↓
                                                          에피소드 내 기능
                                                          • Hand Skip
                                                          • Best Hands 점프


💡 **Best Hands**: 별도 페이지가 아닌, 에피소드 시청 중 해당 에피소드의 Best Hands로 점프하는 기능입니다.

## 2. 홈페이지 구조


### 2.0 디자인 의도 및 목적

**핵심 목표:** 포커 VOD 시청에 최적화된 몰입형 경험 제공

**디자인 결정 및 의도:**

• **다크 테마**
  장시간 시청 시 눈의 피로 감소, 영상 콘텐츠 집중도 향상

• **Netflix 스타일 레이아웃**
  검증된 VOD 플랫폼 UX 패턴으로 학습 비용 최소화

• **가로 스크롤 카드**
  많은 콘텐츠를 공간 효율적으로 탐색, 모바일 제스처 친화적

• **Continue Watching 상단 배치**
  재방문 사용자의 즉시 시청 재개 유도 (리텐션 핵심)

• **시리즈 그룹핑**
  WSOP/HCL 등 시리즈별 정주행 유도, 시청 시간 증가


**사용자 시나리오별 설계:**

┌─────────────────────┬──────────────────────┬─────────────────────┐
│ 시나리오             │ 지원 요소             │ 기대 효과            │
├─────────────────────┼──────────────────────┼─────────────────────┤
│ "이어서 볼래"        │ Continue Watching    │ 1클릭 재생 재개      │
│ "새로운 거 뭐 있지?" │ Recently Added       │ 신규 콘텐츠 발견     │
│ "정주행 하고 싶어"   │ Series Section       │ 시리즈 단위 탐색     │
│ "뭐 볼지 모르겠어"   │ Hero Banner 자동 추천 │ 결정 장애 해소       │
└─────────────────────┴──────────────────────┴─────────────────────┘

## 3. 콘텐츠 카드 디자인

📎 **[이미지 삽입: 08-content-cards.png]**


### 3.0 디자인 의도 및 목적

**핵심 목표:** 콘텐츠 정보를 빠르게 파악하고 시청 결정을 돕는 카드 시스템

**디자인 결정 및 의도:**

• **16:9 썸네일** → 영상 콘텐츠 표준 비율, 실제 장면 미리보기 제공
• **진행률 바** → 이어보기 위치 즉시 파악, 재시청 동기 부여
• **메타 정보 표시** → 핸드 수/액션 시간으로 콘텐츠 밀도 예측
• **배지 시스템** → NEW/4K/HD 등 콘텐츠 특성 즉시 인지
• **호버 확대** → 관심 콘텐츠 강조, 클릭 유도


### 3.1 에피소드 카드

기본 콘텐츠 단위로, 하나의 완전한 에피소드를 표현합니다.

    ┌─────────────────────────────────┐
    │  [썸네일 16:9]          2:34:56 │
    │                                 │
    │  WSOP 2024                      │
    │  Main Event Day 5               │
    │  47 Hands / 52m Action          │
    └─────────────────────────────────┘

**호버 동작:**
• scale(1.05) 확대
• box-shadow 추가
• 재생 버튼 오버레이 표시


### 3.2 Best Hand 카드

에피소드 내 하이라이트 핸드를 표현합니다.

    ┌─────────────────────────────────┐
    │  [All-in] ▶ 2:34                │
    │  [썸네일 16:9]                  │
    │                                 │
    │  AA vs KK                       │
    │  Phil Ivey vs Tom Dwan          │
    │                        $2.3M Pot│
    └─────────────────────────────────┘

**클릭 동작:**
• 해당 에피소드의 타임스탬프로 직접 점프


### 3.3 이어보기 카드

사용자가 시청 중인 콘텐츠를 표현합니다.

    ┌─────────────────────────────────┐
    │  [썸네일 16:9]                  │
    │         ▶ Resume                │
    │  ████████░░░░░░░░░░░░░░░ 45%    │
    │  Main Event Day 5               │
    │  45:23 remaining                │
    └─────────────────────────────────┘

**동작 정의:**
• 클릭 시 중단 지점에서 즉시 재생
• X 버튼으로 목록에서 제거 가능
• 95% 이상 시청 시 자동 제거


### 3.4 콘텐츠 상태 배지

콘텐츠 특성을 시각적으로 구분합니다.

• **NEW** (빨강 #e50914) - 7일 이내 추가
• **4K** (금색 #ffc107) - 4K 리마스터 콘텐츠
• **HD** (회색 #666) - 1080p 콘텐츠
• **CC** (흰색 테두리) - 자막 지원

**4K Remaster 특별 표시:**
• 썸네일 상단에 "✨ 4K REMASTERED" 배너
• 금색 테두리 강조

💡 **참고**: WSOPTV는 구독자 전용 서비스입니다. 무료 콘텐츠(쇼츠, 하이라이트)는 YouTube에서 제공합니다.

## 4. 구독 전환 UX

📎 **[이미지 삽입: 09-subscription.png]**


### 4.0 디자인 의도 및 목적

**핵심 목표:** 비구독자를 자연스럽게 구독자로 전환

**디자인 결정 및 의도:**

• **소프트 Paywall** → 콘텐츠 일부 노출로 관심 유발 후 구독 유도
• **혜택 중심 메시지** → 가격보다 가치를 먼저 전달
• **연간 플랜 강조** → LTV 극대화, 이탈률 감소
• **원클릭 결제** → Apple Pay/Google Pay로 전환 장벽 최소화
• **미리보기 제공** → 30초 미리보기로 콘텐츠 품질 확인 기회

**전환 퍼널:**

    1. 인지 ──── 홈 브라우징 ──────── 잠금 콘텐츠 발견
         ↓
    2. 관심 ──── 잠금 콘텐츠 클릭 ──── 30초 미리보기 시청
         ↓
    3. 결정 ──── 미리보기 종료 ─────── Paywall 모달 표시
         ↓
    4. 전환 ──── CTA 클릭 ─────────── 구독 페이지 이동


### 4.1 Paywall 화면

비구독자가 유료 콘텐츠 클릭 시 표시됩니다.

    ┌─────────────────────────────────────────────┐
    │         [블러 처리된 콘텐츠 썸네일]           │
    │                                             │
    │                   🔒                        │
    │                                             │
    │    "이 콘텐츠는 Premium 구독자 전용입니다"    │
    │                                             │
    │    ✓ 전체 아카이브 (18TB+)                  │
    │    ✓ Hand Skip                             │
    │    ✓ Best Hands                            │
    │    ✓ 이어보기 동기화                        │
    │    ✓ 광고 없음                              │
    │                                             │
    │         [ Premium 시작하기 ]                │
    │                                             │
    │         $9.99/월 또는 $99/년                │
    └─────────────────────────────────────────────┘

**동작:**
• ESC 또는 배경 클릭으로 닫기
• "Premium 시작하기" → 구독 페이지 이동
• 30초 미리보기 후 자동 표시


### 4.2 구독 페이지

플랜 선택 및 결제를 처리합니다.

**플랜 비교:**

    ┌──────────────────┐  ┌──────────────────────────┐
    │     Monthly      │  │   Yearly ⭐ BEST VALUE   │
    ├──────────────────┤  ├──────────────────────────┤
    │    $9.99/월      │  │  $99/년 ($8.25/월)       │
    │                  │  │     17% 절약             │
    │    매월 결제     │  │     매년 결제            │
    └──────────────────┘  └──────────────────────────┘

**결제 수단:**
• 💳 Credit Card (기본)
• 🍎 Apple Pay (iOS 우선 표시)
• G Google Pay (Android 우선 표시)

**페이지 구조:**
1. 헤더: "WSOPTV Premium 구독"
2. 플랜 선택 카드 (Monthly / Yearly)
3. 혜택 요약 리스트
4. 결제 수단 선택
5. 결제 버튼
6. 약관 동의 체크박스

**전환 최적화:**
• Yearly 플랜 기본 선택
• "가장 인기" 배지로 사회적 증거
• 환불 정책 명시 (7일 이내)

### 5.0 디자인 의도 및 목적

**핵심 목표:** 콘텐츠 탐색과 검색을 하나의 통합된 경험으로 제공

**디자인 결정 및 의도:**

• **통합 페이지** → 검색과 브라우징을 분리하지 않고 컨텍스트 유지
• **상태 기반 UI** → 검색 활성화 여부에 따라 최적화된 레이아웃 표시
• **필터 사이드바** → 시리즈/연도/언어별 빠른 필터링
• **실시간 검색** → 타이핑 중 자동완성으로 탐색 시간 단축

## 6. 플레이어 페이지

📎 **[이미지 삽입: 13-player.png]**


### 6.0 디자인 의도 및 목적

**핵심 목표:** 포커 VOD에 최적화된 시청 경험 제공

**디자인 결정 및 의도:**

• **Hand Skip** → 폴드/탱킹 구간 자동 스킵으로 액션만 시청
• **Best Hands 패널** → 에피소드 내 하이라이트 빠른 접근
• **핸드 마커** → 프로그레스 바에 핸드 시작점 시각화
• **키보드 중심** → N키로 다음 핸드, B키로 Best Hands

## 7. 계정 페이지

📎 **[이미지 삽입: 14-account.png]**


### 7.0 디자인 의도 및 목적

**핵심 목표:** 계정 관리 기능을 직관적인 탭 구조로 제공

**디자인 결정 및 의도:**

• **사이드바 네비게이션** → 섹션별 빠른 접근
• **프로필 우선** → 가장 자주 사용하는 정보 상단 배치
• **구독 상태 강조** → 현재 플랜과 갱신일 명확히 표시
• **시청 기록 통합** → 이어보기와 시청 완료 구분


### 7.1 사이드바 메뉴

    ┌───────────────┐
    │ 👤 프로필     │ ← 사용자 정보 관리
    │ 💳 구독 관리  │ ← 플랜, 결제 정보
    │ 📺 시청 기록  │ ← 이어보기, 시청 완료
    │ ⚙️ 설정       │ ← 재생, 자막, 알림
    └───────────────┘


### 7.2 각 섹션 상세

**프로필**
• 프로필 이미지 - 변경 가능
• 이름 - 편집 가능
• 이메일 - 표시 (변경 시 인증 필요)
• 비밀번호 - 변경 버튼

**구독 관리**
• 현재 플랜 - Premium / Free
• 다음 결제일 - 날짜 표시
• 결제 수단 - 마스킹된 카드 정보
• 업그레이드/해지 - 액션 버튼

**시청 기록**
• 이어보기 탭 - 진행 중인 콘텐츠
• 시청 완료 탭 - 완료된 콘텐츠

**설정**
• 재생 - 자동 재생, 품질, Hand Skip 기본값
• 자막 - 언어, 크기, 스타일
• 알림 - 새 콘텐츠, 이메일 수신

### 8.2 회원가입

**단계:**
1. 기본 정보 - 이름, 이메일
2. 보안 - 비밀번호 (강도 표시), 비밀번호 확인

• 최소 8자
• 대문자, 소문자, 숫자 포함
