## 1. 사용자 여정 (User Journey)

### 1.1 신규 사용자 여정

**진입 경로 1: YouTube (신규 유저층 확보)**

```mermaid
flowchart LR
    A[YouTube 접속] --> B[쇼츠/하이라이트]
    B --> C{풀 영상 관심}
    C -->|Yes| D[WSOPTV 이동]
    D --> E[구독]
```

**진입 경로 2: WSOPTV 직접 접속**

```mermaid
flowchart LR
    A[랜딩페이지 접속] --> B[콘텐츠 미리보기]
    B --> C[회원가입] --> D[구독]
```

> **투트랙 전략**: YouTube는 신규 유저층이 많아 무료 콘텐츠(쇼츠, 하이라이트)로 유입을 유도합니다. WSOPTV는 구독자 전용 풀 에피소드 서비스입니다.

### 1.2 기존 사용자 여정

```mermaid
flowchart TB
    A[앱/웹 접속] --> B[로그인]
    B --> C[홈 화면]

    C --> D{시청 선택}

    D -->|이어보기| E[중단 지점부터 재생]
    D -->|새 콘텐츠| F[브라우징]

    F --> G[에피소드 선택]
    G --> H[시청 시작]

    H --> I{에피소드 내 기능}
    I --> J[Hand Skip]
    I --> K[Best Hands 점프]
```

> **Best Hands**: 별도 페이지가 아닌, 에피소드 시청 중 해당 에피소드의 Best Hands로 점프하는 기능입니다.

## 2. 홈페이지 구조

### 2.0 디자인 의도 및 목적

**핵심 목표:** 포커 VOD 시청에 최적화된 몰입형 경험 제공

| 디자인 결정 | 의도 |
|-------------|------|
| **다크 테마** | 장시간 시청 시 눈의 피로 감소, 영상 콘텐츠 집중도 향상 |
| **Netflix 스타일 레이아웃** | 검증된 VOD 플랫폼 UX 패턴으로 학습 비용 최소화 |
| **가로 스크롤 카드** | 많은 콘텐츠를 공간 효율적으로 탐색, 모바일 제스처 친화적 |
| **Continue Watching 상단 배치** | 재방문 사용자의 즉시 시청 재개 유도 (리텐션 핵심) |
| **시리즈 그룹핑** | WSOP/HCL 등 시리즈별 정주행 유도, 시청 시간 증가 |

**사용자 시나리오별 설계:**

| 시나리오 | 지원 요소 | 기대 효과 |
|----------|-----------|-----------|
| "이어서 볼래" | Continue Watching | 1클릭 재생 재개 |
| "새로운 거 뭐 있지?" | Recently Added + NEW 배지 | 신규 콘텐츠 발견 |
| "2024 메인이벤트 정주행" | Series Section | 시리즈 단위 탐색 |
| "뭐 볼지 모르겠어" | Hero Banner 자동 추천 | 결정 장애 해소 |

## 3. 콘텐츠 카드 디자인

![Content Cards](../../../docs/wireframes/v2/08-content-cards.png)

### 3.0 디자인 의도 및 목적

**핵심 목표:** 콘텐츠 정보를 빠르게 파악하고 시청 결정을 돕는 카드 시스템

| 디자인 결정 | 의도 |
|-------------|------|
| **16:9 썸네일** | 영상 콘텐츠 표준 비율, 실제 장면 미리보기 제공 |
| **진행률 바** | 이어보기 위치 즉시 파악, 재시청 동기 부여 |
| **메타 정보 표시** | 핸드 수/액션 시간으로 콘텐츠 밀도 예측 |
| **배지 시스템** | NEW/4K/HD 등 콘텐츠 특성 즉시 인지 |
| **호버 확대** | 관심 콘텐츠 강조, 클릭 유도 |

---

### 3.1 에피소드 카드

기본 콘텐츠 단위로, 하나의 완전한 에피소드를 표현합니다.

| 요소 | 스펙 | 설명 |
|------|------|------|
| 썸네일 | 16:9, WebP, Lazy Load | 에피소드 대표 장면 |
| 재생시간 | 우하단, 반투명 배경 | `H:MM:SS` 형식 |
| 시리즈명 | 상단, 작은 텍스트 | WSOP 2024, HCL S2 등 |
| 제목 | 중앙, 볼드 | Main Event Day 5 |
| 메타 정보 | 하단, 회색 텍스트 | `47 Hands / 52m Action` |

**호버 동작:**
- `scale(1.05)` 확대
- `box-shadow` 추가
- 재생 버튼 오버레이 표시

---

### 3.2 Best Hand 카드

에피소드 내 하이라이트 핸드를 표현합니다.

| 요소 | 스펙 | 설명 |
|------|------|------|
| 썸네일 | 16:9, 해당 핸드 장면 | 올인/쇼다운 순간 캡처 |
| 액션 배지 | 좌상단, #e50914 | All-in, Bluff, Bad Beat 등 |
| 타임스탬프 | 배지 옆 | `▶ 2:34` (점프 위치) |
| 핸드 제목 | 중앙 | AA vs KK |
| 플레이어 | 하단 | Phil Ivey vs Tom Dwan |
| 팟 사이즈 | 우하단, 금색 | $2.3M Pot |

**클릭 동작:**
- 해당 에피소드의 타임스탬프로 직접 점프

### 3.3 이어보기 카드

사용자가 시청 중인 콘텐츠를 표현합니다.

| 요소 | 스펙 | 설명 |
|------|------|------|
| 썸네일 | 16:9, 현재 장면 | 중단 시점 프레임 |
| 재생 버튼 | 중앙 오버레이 | ▶ Resume |
| 진행률 바 | 하단, 4px, #e50914 | 시청 진행률 |
| 제목 | 카드 하단 | Main Event Day 5 |

**동작 정의:**
- 클릭 시 중단 지점에서 즉시 재생
- X 버튼으로 목록에서 제거 가능
- 95% 이상 시청 시 자동 제거

### 3.4 콘텐츠 상태 배지

콘텐츠 특성을 시각적으로 구분합니다.

| 배지 | 색상 | 조건 |
|------|------|------|
| **NEW** | #e50914 (빨강) | 7일 이내 추가 |
| **4K** | #ffc107 (금색) | 4K 리마스터 콘텐츠 |

## 4. 구독 전환 UX

![Subscription UX](../../../docs/wireframes/v2/09-subscription.png)

### 4.0 디자인 의도 및 목적

**핵심 목표:** 비구독자를 자연스럽게 구독자로 전환

| 디자인 결정 | 의도 |
|-------------|------|
| **소프트 Paywall** | 콘텐츠 일부 노출로 관심 유발 후 구독 유도 |
| **혜택 중심 메시지** | 가격보다 가치를 먼저 전달 |
| **연간 플랜 강조** | LTV 극대화, 이탈률 감소 |
| **원클릭 결제** | Apple Pay/Google Pay로 전환 장벽 최소화 |
| **미리보기 제공** | 30초 미리보기로 콘텐츠 품질 확인 기회 |

**전환 퍼널:**

| 단계 | 트리거 | 액션 |
|------|--------|------|
| 1. 인지 | 홈 브라우징 | 잠금 콘텐츠 발견 |
| 2. 관심 | 잠금 콘텐츠 클릭 | 30초 미리보기 시청 |
| 3. 결정 | 미리보기 종료 | Paywall 모달 표시 |
| 4. 전환 | CTA 클릭 | 구독 페이지 이동 |

---

### 4.1 Paywall 화면

비구독자가 유료 콘텐츠 클릭 시 표시됩니다.

**레이아웃:**

| 영역 | 내용 |
|------|------|
| 배경 | 클릭한 콘텐츠 썸네일 (블러 처리) |
| 아이콘 | 🔒 자물쇠 |
| 메시지 | "이 콘텐츠는 Premium 구독자 전용입니다" |
| 혜택 목록 | 5개 핵심 혜택 |
| CTA | "Premium 시작하기" 버튼 |
| 가격 | $9.99/월 또는 $99/년 |

**Premium 혜택:**

| 혜택 | 설명 |
|------|------|
| 전체 아카이브 | 18TB+ WSOP/HCL/GGPoker 콘텐츠 |
| Hand Skip | 액션 구간만 빠르게 시청 |
| Best Hands | 에피소드별 하이라이트 점프 |
| 이어보기 동기화 | 모든 기기에서 시청 위치 유지 |
| 광고 없음 | 끊김 없는 시청 경험 |

**동작:**
- ESC 또는 배경 클릭으로 닫기
- "Premium 시작하기" → 구독 페이지 이동
- 30초 미리보기 후 자동 표시

---

### 4.2 구독 페이지

플랜 선택 및 결제를 처리합니다.

**플랜 비교:**

| 항목 | Monthly | Yearly (추천) |
|------|---------|---------------|
| 가격 | $9.99/월 | $99/년 ($8.25/월) |
| 할인 | - | 17% 절약 |
| 결제 주기 | 매월 | 매년 |
| 강조 | - | ⭐ BEST VALUE 배지 |

**결제 수단:**

| 수단 | 아이콘 | 우선순위 |
|------|--------|----------|
| Credit Card | 💳 | 기본 |
| Apple Pay | 🍎 | iOS 우선 표시 |
| Google Pay | G | Android 우선 표시 |

**페이지 구조:**
1. 헤더: "WSOPTV Premium 구독"
2. 플랜 선택 카드 (Monthly / Yearly)
3. 혜택 요약 리스트
4. 결제 수단 선택
5. 결제 버튼
6. 약관 동의 체크박스

**전환 최적화:**
- Yearly 플랜 기본 선택
- "가장 인기" 배지로 사회적 증거
- 환불 정책 명시 (7일 이내)

## 5. Browse & Search 페이지

![Browse Page](../../../docs/wireframes/v2/12-browse.png)

### 5.0 디자인 의도 및 목적

**핵심 목표:** 콘텐츠 탐색과 검색을 하나의 통합된 경험으로 제공

| 디자인 결정 | 의도 |
|-------------|------|
| **통합 페이지** | 검색과 브라우징을 분리하지 않고 컨텍스트 유지 |
| **상태 기반 UI** | 검색 활성화 여부에 따라 최적화된 레이아웃 표시 |
| **필터 사이드바** | 시리즈/연도/언어별 빠른 필터링 |
| **실시간 검색** | 타이핑 중 자동완성으로 탐색 시간 단축 |

---

### 5.1 기본 상태 (Grid View)

검색어 없이 페이지 접근 시 기본 레이아웃입니다.

| 영역 | 내용 |
|------|------|
| 검색 바 | 상단 고정, 플레이스홀더: "시리즈, 플레이어, 이벤트 검색..." |
| 필터 사이드바 | 좌측, 시리즈/연도/언어 필터 |
| 콘텐츠 그리드 | 4열 카드 그리드 |
| 정렬 옵션 | 최신순, 인기순, 제목순 |

**필터 카테고리:**

| 필터 | 옵션 |
|------|------|
| 시리즈 | WSOP, HCL, GGPoker, PAD, MPP 등 |
| 연도 | 2024, 2023, 2022... |
| 언어 | 영어, 한국어 자막 |
| 품질 | 4K, HD |

---

### 5.2 검색 활성 상태 (List View)

검색어 입력 시 리스트 뷰로 전환됩니다.

| 영역 | 내용 |
|------|------|
| 자동완성 | 최대 5개 추천 결과 |
| 검색 결과 수 | "'검색어' 검색 결과: N개" |
| 결과 목록 | 리스트 형식, 상세 정보 표시 |

**검색 결과 항목:**

| 요소 | 설명 |
|------|------|
| 썸네일 | 좌측 고정 (16:9) |
| 제목 | 검색어 하이라이트 |
| 시리즈/연도 | 메타 정보 |
| 설명 | 2줄 요약 |
| 재생 시간 | 우측 표시 |

## 6. 플레이어 페이지

![Player Page](../../../docs/wireframes/v2/13-player.png)

### 6.0 디자인 의도 및 목적

**핵심 목표:** 포커 VOD에 최적화된 시청 경험 제공

| 디자인 결정 | 의도 |
|-------------|------|
| **Hand Skip** | 폴드/탱킹 구간 자동 스킵으로 액션만 시청 |
| **Best Hands 패널** | 에피소드 내 하이라이트 빠른 접근 |
| **핸드 마커** | 프로그레스 바에 핸드 시작점 시각화 |
| **키보드 중심** | N키로 다음 핸드, B키로 Best Hands |

---

### 6.1 플레이어 컨트롤

**기본 컨트롤:**

| 요소 | 동작 |
|------|------|
| 재생/일시정지 | 중앙 대형 버튼 |
| 프로그레스 바 | 드래그로 탐색, 핸드 마커 표시 |
| 볼륨 | 슬라이더 + 음소거 버튼 |
| 전체화면 | 우측 버튼 |

**포커 전용 컨트롤:**

| 요소 | 상태 | 설명 |
|------|------|------|
| Hand Skip | ON/OFF | 다음 핸드까지 남은 시간 표시 |
| Best Hands | ON/OFF | 하이라이트 개수 표시 |

---

### 6.2 Best Hands 패널

| 요소 | 설명 |
|------|------|
| 핸드 카드 | 썸네일 + 액션 배지 + 타임스탬프 |
| 플레이어 정보 | 대결 구도 (vs) |
| 팟 사이즈 | 금색 강조 |
| 클릭 동작 | 해당 시점으로 점프 |

**키보드 단축키:**

| 키 | 동작 |
|---|------|
| Space | 재생/일시정지 |
| N | 다음 핸드로 스킵 |
| B | Best Hands 패널 토글 |
| F | 전체화면 |
| M | 음소거 |
