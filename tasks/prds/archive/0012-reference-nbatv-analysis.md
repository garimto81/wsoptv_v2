# NBA League Pass / NBA TV - 레퍼런스 분석

**문서 번호**: REF-0012 | **버전**: 2.0.0 | **작성일**: 2025-12-17

---

## 1. 서비스 개요

| 항목 | NBA League Pass | NBA TV | NBA App (무료) |
|------|-----------------|--------|---------------|

## 3. 무료 티어 핵심 기능

### 3.1 NBA CrunchTime (핵심)

**모든 경기의 클러치 타임 무료 라이브 시청**

```
┌─────────────────────────────────────────────────────────────┐
│  NBA CrunchTime - 무료                                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  • 모든 경기 마지막 몇 분 무료 라이브 시청                   │
│  • "Whiparound" 방식 - 박빙 경기 실시간 전환                │
│  • NBA App 독점 (NBA TV 동시 방송)                         │
│  • 호스트: Jared Greenberg                                  │
│                                                             │
│  → WSOPTV 적용: 베스트 핸드 일부 무료 제공                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 NBA Vault (클래식 아카이브)

- 역대 명경기 무료 시청
- 레전드 경기 아카이브 접근

### 3.3 Moments

- 세로형 소셜 스타일 피드
- 빠른 하이라이트 소비
- Roku TV 지원

## 4. 핵심 기술 기능 (WSOPTV 참고)

### 4.1 Condensed Game (압축 경기)

**Full Game을 다양한 길이로 압축 제공**

| 버전 | 길이 | 내용 |
|------|------|------|
| **Full Game** | ~90분 | 전체 경기 (광고 제거) |
| **All Possessions** | ~30분 | 모든 공격 장면 |
| **10-Minute Condensed** | 10-15분 | 득점/하이라이트만 |
| **Key Highlights** | ~5분 | 핵심 플레이만 |

```
┌─────────────────────────────────────────────────────────────┐
│  Game Details 화면                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  LAL vs GSW - December 15, 2024                            │
│                                                             │
│  시청 옵션:                                                 │
│  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐       │
│  │ Full Game    │ │ Condensed    │ │ Highlights   │       │
│  │ 2h 15m       │ │ 12m          │ │ 5m           │       │
│  └──────────────┘ └──────────────┘ └──────────────┘       │
│                                                             │
│  → WSOPTV 적용: Hand Skip (52분) / Best Hands (18분)       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.2 Smart Rewind (스마트 되감기)

**게임 중간부터 시청 시 핵심 플레이 자동 표시**

```
┌─────────────────────────────────────────────────────────────┐
│  Smart Rewind 기능                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  타임라인:                                                  │
│  ════════════════════════════════════════════════════════   │
│        ▲      ▲           ▲        ▲              ▲        │
│       Q1     Q2          Q3       Q4            OT        │
│       던크   3점슛       블록     클러치        결승골     │
│                                                             │
│  • 어느 시점에서든 키 플레이 하이라이트 표시                │
│  • 원하는 플레이로 즉시 점프 가능                           │
│                                                             │
│  → WSOPTV 적용: Best Hands 타임스탬프 점프와 동일 개념     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.3 AI 기반 기능 (2024-25 신규)

| 기능 | 기술 | 설명 |
|------|------|------|
| **NBA Insights** | Microsoft Azure AI | 실시간 선수 스탯, 마일스톤 알림 |
| **Dunk Score** | AI 계산 | 덩크 점수 (점프 거리, 스타일, 수비 등) |
| **Synced Stats** | 실시간 연동 | 경기 시청 중 실시간 통계 |

```
┌─────────────────────────────────────────────────────────────┐
│  NBA Insights 예시                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🔥 LeBron James가 40,000점 달성까지 12점 남았습니다       │
│  📊 Stephen Curry 3점슛 성공률: 오늘 7/10 (70%)            │
│  ⚡ 이번 쿼터 Lakers 득점: 34점 (시즌 최고)                │
│                                                             │
│  → WSOPTV 향후 적용: 핸드 분석 AI, 플레이어 인사이트       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.4 Multiview (4분할 화면)

- League Pass 구독자 전용
- 최대 4개 경기 동시 시청
- Connected TV (Roku, Apple TV 등) 지원

## 5. WSOPTV vs NBA League Pass 비교

### 5.1 기능 매핑

| NBA 기능 | WSOPTV 대응 | 차이점 |
|----------|------------|--------|
| **Condensed Game** | **Best Hands** | NBA: 득점 중심 / WSOP: 베스트 핸드 |
| **Smart Rewind** | **타임스탬프 점프** | 동일 개념 |
| **NBA CrunchTime** | **무료 하이라이트** | 클러치 타임 ↔ 베스트 핸드 |
| **NBA Vault** | **클래식 시즌 무료** | 역대 명경기 아카이브 |
| **Multiview** | - | 포커는 단일 테이블 집중 |
| **AI Insights** | - | 향후 확장 (핸드 분석 AI) |
| **Dunk Score** | - | 향후 확장 (블러프 스코어?) |

### 5.3 콘텐츠 압축 비교

| 구분 | NBA | WSOPTV |
|------|-----|--------|
| **원본 길이** | ~2.5시간 (경기) | 5-10시간+ (하루 방송) |
| **압축 버전 1** | Condensed (10-15분) | Best Hands (30-45분) |
| **압축 버전 2** | Key Highlights (5분) | (향후 추가 가능) |
